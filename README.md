# Парсер сайта https://byrutgame.org/ на NodeJS.
### Парсер https://byrutgame.org/ с использованием библиотеки cheerio, axios и сохранением данных в .txt файл. 

### Парсер работает следующим образом:
  1. Мы получаем страницу https://byrutgame.org/top-torrent-games/
  2. Ищем все теги <a>, которые находятся в <div> с классом pages и извлекаем текст
  3. Далее с помощью цикла проходимся по каждой старанице
  4. Ищем все теги с классом short_item, в котором ищем <div> теги с классом short_title и извлекаем текст. Также ищем все теги <a> и извлекаем ссылки
  5. Далее созраняем в .txt файле

```bash
Клонируем репозиторий
https://github.com/KellerHalm/NodeJS-Parse.git

Устанавливаем зависимости
npm install

Запускаем
node index.js
```

> после запуска все данные сохраняеются в data.txt
